Seller Registration Request on Portal Odoo Apps

Website Vendor Registration Odoo App simplifies a vendor onboarding workflow by allowing users to submit registration requests on a website. Purchase managers gain a centralized view of all requests, equipped to approve or reject vendors quickly and efficiently. Additionally, the rejection process requires the entry of a rejection reason, ensuring transparent communication. To enhance user engagement, automated email notifications are triggered upon the creation or rejection of vendor requests, improving a seamless and communicative vendor management experience.

Features

Vendor Registration Request through Website

User can create vendor registration request directly from website.

Display Vendor Registration Request in Purchase

User can view all the registration request of vendor in purchase order.

Approve/Reject Vendor Registration Request

Purchase manager has access to approve or reject the vendor registration request.

Email Notification for Approve/Reject Vendor

When creation of vendor or rejection of vendor, User will get email notification for that.
Demo Request Video Tutorial

Vendor Registration Request Menu in Website

User can click on "Vendor Registration Request" menu under "Vendor Request" menu in website.

Vendor Registration Request Form

In vendor registration request form, User can select vendor type as "Individual" or "Company" and enter details of vendor then click on "Vendor Registration Request" button.

Successfully Created Vendor Registration Request

Vendor Registration Request in Purchase

Goto Purchase -> Orders -> Vendor Registration Request menu in that view the requests of vendor registration.

Create Vendor

User can view vendor registration request in form view then they can create or reject the vendor request and stage in "TO APPROVE".

User can create vendor by clicking on "CREATE VENDOR" button.

Approved Vendor Registration Request

Once user can click on "CREATE VENDOR" button, Stage moved in "APPROVED" and view the "Vendor" smart button to show the vendor.

Email Notification for Vendor Creation Accepted

Reject Vendor

User can view vendor registration request in form view then they can create or reject the vendor request and stage in "TO APPROVE".

User can reject vendor by clicking on "REJECT" button.

Vendor Registration Reason Wizard

When user reject the vendor, Must be enter "Reject Reason" and click on "CONFIRM" button.

Rejected Vendor Registration Request

Once user entered reject reason and confirm it, Stage moved in "REJECTED".

Email Notification for Reject the Vendor Request
